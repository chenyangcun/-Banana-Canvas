# AI Studio中运行

AI Studio演示地址: https://ai.studio/apps/drive/1I8vIZhBkAZs9mzhRb7aPeXhAEJ09EX9o

## 本地运行

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
